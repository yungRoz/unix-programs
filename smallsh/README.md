# Smallsh

A shell program written in C!

## Overview

Supports Unix commands, standard input/output redirection, background/foreground processes, comments. Some commands built-in, they include cd, status, and exit.

### Cool stuff

Mutex-based time keeping via read/write operations and multithreads by the pthread library.

### Run

```
$ gcc -o smallsh smallsh.c
```
